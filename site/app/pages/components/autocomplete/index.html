Input text can be used with autocomplete to help users who have limited literacy or who write in a foreign language. For example, autocomplete can suggest input as it’s typed (refreshing suggestions with each keystroke).

Jinge Material autocomplete is really simple, yet powerfull. With simple options you can create great suggestions with async feedbacks.

Custom Template

Autocomplete also accepts a custom template, flexible to accept any HTML element and with an 'empty state' built in. You can also highlight the search term inside the matches, to give a feedback on why that item has been in the results. Awesome:

Although the md-highlight component is most used with autocomplete, you can use it anywhere.

Search Algorithms

Jinge Material autocomplete comes with 3 ways of search: Fuzzy search , search by includes, search by starts. Default is fuzzy search. The fuzzy search tries to match the results by approximation, finding patterns inside the available options. This will help with accidental type errors and improve the results. If you think that this may be confusing, you can disable this. Example:
