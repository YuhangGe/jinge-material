Material icons uses geometric shapes to visually represent core ideas, capabilities, or topics. They are generally used along with the Material icons

<_t>Material 图标

在模板中，可以使用 <md-icon-{THEME}-{ICON_ID}> 这个格式的组件别名来引用图标组件。 模板中的 {THEME} 替换为实际需要的主题， 可选的主题为 baseline, outline, round, sharp, twotone 这五个之一， 依次对应 Material Desgin 官网上的 Filled, Outlined, Rounded, Sharp, Two-Tone 主题。 模板中的 {ICON_ID} 替换为实际的图标的 id，比如 home, 3d_rotation 等， 更多的图标 id 请在使用时直接参考 Material Desgin 官网。

通常情况下，一个项目只会使用一个默认的主题。因此，还可以直接使用 <md-icon-{ICON_ID}> 这个格式的组件别名来引用图标组件。 默认的主题为 baseline，但可以在 webpack 的配置中指定其它主题为默认主题。

<_t>Icon Font 图标

直接使用 <md-icon> 组件，可以使用 Icon Font 图标库，比如 Font Awesome。 需要自己引入相应的 Web Font 资源。

<_t>SVG 图标

如果给 <md-icon> 组件指定 <src> 属性， 组件会调用 fetch 函数来加载该指定的远程图片，加载完后渲染到组件内部。
