Menus appear upon interaction with a button, action, or other control. They usually display a list of choices, with one choice per line, but can also show a rich content.

Placement

Default placement is bottom, support all popper.js placements

Offset

By default menu will have offset relative to trigger. If it is not what we want, we can simple set offset="0" or set some custom offset.
